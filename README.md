## 制作背景

旅行中や友人との食事・遊びの際に、誰かが一時的に費用を立て替える場面は多くあります。しかし、後から「誰が何円払ったのか」「誰が誰にいくら返すべきなのか」を整理するのは手間がかかり、記録漏れや計算ミスが発生しやすいと感じていました。

そこで、レシートを撮影するだけで支払い内容を記録し、最終的に誰が誰にいくら支払えばよいのかを簡単に確認できる割り勘アプリ「ワリポケ」を開発しました。精算にかかる手間を減らし、友人同士での金銭のやり取りをよりスムーズにすることを目的としています。
